# DL-Learning-Log
## 简介
个人深度学习系统学习仓库，收纳学习笔记、实战代码、数据集脚本、实验记录与论文研读内容，基于PyTorch完成模型复现、算法练习与项目实战，持续更新迭代。

## 仓库目录
```
├── Notes/        # 理论知识点笔记、公式推导、学习总结
├── Code/         # 深度学习实战代码、经典网络实现、训练demo
├── Dataset/      # 常用数据集下载脚本、数据预处理工具
├── Experiment/   # 训练实验日志、调参记录、结果分析
├── Paper/        # 论文阅读笔记、论文复现代码
├── Resources/    # 环境配置、学习资料、实用工具教程
```

## 环境依赖
```
torch
torchvision
numpy
matplotlib
opencv-python
scikit-learn
```

## 学习内容
1. 深度学习基础：神经网络、激活函数、损失函数、反向传播
2. 计算机视觉：CNN系列模型、图像分类、数据增强
3. 时序模型：RNN、LSTM基础原理与实战
4. 进阶学习：Transformer、模型调优、训练技巧
5. 项目实战：小型深度学习案例复现与训练

## 使用说明
1. 运行`Dataset`内脚本可自动下载公开数据集
2. 所有代码均可本地直接运行，适配主流CUDA训练环境
3. 数据集、模型权重、日志文件已加入忽略，不上传大文件
4. 笔记与代码同步更新，记录学习过程与踩坑经验
